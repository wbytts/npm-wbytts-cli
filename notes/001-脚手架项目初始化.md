
# 初始化项目

npm init -y
npm install lerna -D

编辑 package.json
```json
{
  "name": "npm-wbytts-cli",
  "private": true,
  "workspaces": [
    "packages/*"
  ],
  "version": "0.0.1",
  "description": "冰冰的脚手架练习",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "devDependencies": {
    "lerna": "^8.1.9"
  }
}
```

# 使用 lerna 初始化项目

初始化 lerna 项目：`lerna init`

这个命令执行后，生成了一个 lerna 的配置文件 `lerna.json` 

# 使用 lerna 创建新的包

```
npx lerna create @wbytts/cli
```
